Personal Finance Dashboard
A full-stack finance management dashboard built using React, Tailwind CSS, Node.js, Express, and MongoDB.
Features
    • Authentication with JWT
    • Income & Expense Tracking
    • Interactive Charts
    • Monthly Budget Tracking
    • CSV Export
    • Dark Mode
    • Responsive Design
Tech Stack
    • React
    • Tailwind CSS
    • Node.js
    • Express.js
    • MongoDB
    • Recharts
