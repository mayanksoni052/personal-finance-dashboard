**Personal Finance Dashboard**

A full-stack finance management dashboard built using React, Tailwind CSS, Node.js, Express, and MongoDB.

**Features**<br>
• Authentication with JWT <br>
• Income & Expense Tracking<br>
• Interactive Charts<br>
• Monthly Budget Tracking<br>
• CSV Export<br>
• Dark Mode<br>
• Responsive Design<br><br>
**Tech Stack**<br>
• React<br>
• Tailwind CSS<br>
• Node.js<br>
• Express.js<br>
• MongoDB<br>
• Recharts<br>
